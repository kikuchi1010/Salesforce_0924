Pardot（Account Engagement）とは・役割・機能・メリット

Pardotとは

  • Salesforceが提供するB2B向けMAツール
  • 見込み顧客の獲得・育成・営業連携を効率化
  • 現在は「Account Engagement」として提供

基本的な役割

  • リード獲得：WebフォームやLPで情報収集
  • リードナーチャリング：メール配信やスコアリング
  • リードスコアリング：行動を点数化し営業へ
  • レポーティング：施策効果を測定

主な機能

  • メールマーケティング：セグメント配信、A/Bテスト
  • LP／フォーム作成：ノーコード作成、自動連携
  • リード管理：スコア＋グレードで特定
  • キャンペーン連携：売上貢献を可視化

メリットと活用ポイント

  • 営業とマーケ部門の連携 → 商談化率UP
  • B2Bの長期的顧客育成に強い
  • Salesforceとシームレス統合

課題・業務の流れ・目先の実施事項

課題

  • Pardotはマーケティングで活用中
  • データ取得済みだが営業効率化や営業ツールとしての活用が課題

業務の流れ

依頼者 → 髙橋（外部ベンダー） → 確認：JMJ / CE：谷口さん

目先の実施事項

  • 坂口さんから営業アタックリストに関する共有・推進が必要
