Pardot（Account Engagement）とは

Pardotとは

  • Salesforceが提供するB2B向けMAツール
  • 見込み顧客の獲得・育成・営業連携を効率化
  • 現在は「Account Engagement」として提供

基本的な役割

  • リード獲得：WebフォームやLPで情報収集
  • リードナーチャリング：メール配信やスコアリング
  • リードスコアリング：行動を点数化し営業へ
  • レポーティング：施策効果を測定

主な機能

  • メールマーケティング：セグメント配信、A/Bテスト
  • LP／フォーム作成：ノーコード作成、自動連携
  • リード管理：スコア＋グレードで特定
  • キャンペーン連携：売上貢献を可視化

メリットと活用ポイント

  • 営業とマーケ部門の連携 → 商談化率UP
  • B2Bの長期的顧客育成に強い
  • Salesforceとシームレス統合

課題・業務の流れ・目先の実施事項

課題

  • Pardot導入、その後の活用について検討中
  • マーケティング側でのデータ取得フローは作成　
  • 営業効率化や営業ツールとしての活用が課題
